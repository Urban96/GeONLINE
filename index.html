1. Upload data

First you need to upload data from your computer. Only geoJSON format is allowed. Once you choose a file, click on UPLOAD button.

2. Edit symbology

After uploading your data, you are able to edit basic symbology of the layer. You can click ADD TO MAP button.

3. Manage layers

Once you added new layer, in the List of layers you can show and hide layers.

4. Operation

There are plenty of operations to choose. Once you choose one, additional information will appear.

5. Download data

You can download new layer by clicking on the download button.

6. Settings

If you want to change settings, click on Settings button

Select operation



Buffer


Creates buffer polygons around input features to a specified distance.
















Points within polygon


Finds Points that fall within (Multi)Polygon(s).














Calculate centroid


Takes one or more features and calculates the centroid using the mean of all vertices. This lessens the effect of small islands and artifacts when calculating the centroid of a set of polygons.












TIN


Takes a set of points and creates a Triangulated Irregular Network , or a TIN for short, returned as a collection of Polygons. These are often used for developing elevation contour maps or stepped heat visualizations.











Calculate area


Takes one or more features and returns their area measure in square meters or kilometers.




Calculate length


Takes a GeoJSON and measures its length in the specified units, (Multi)Point 's distance are ignored.




Simplify


Takes a GeoJSON object and returns a simplified version. Internally uses simplify-js to perform simplification using the Ramer-Douglas-Peucker algorithm.


















Polygon to points


Takes a feature or set of features and returns all positions as points.












Line to polygon


Converts (Multi)LineString(s) to Polygon(s).













Voronoi polygons


Voronoi polygons are created so that every location within a polygon is closer to the sample point in that polygon than any other sample point. After the polygons are created, neighbors of a sample point are defined as any other sample point whose polygon shares a border with the chosen sample point.













Envelope


Takes any number of features and returns a rectangular Polygon that encompasses all vertices.














Create random points


This function returns a random points.















Create random lines


This function returns a random lines.






















Create random polygons


This function returns a random polygons.
